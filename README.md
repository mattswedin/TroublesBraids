# TroublesBraids

# About

Trouble's Braids is a salon in Brooklyn, NY. With this Web App, you can book appointments, view stylists, buy products, apply for chair rental, and more.

# Core Technologies

**Frontend:** React for a dynamic user interface.<br>
**Backend:** ASP.NET Core for RESTful APIs or GraphQL APIs.<br>
**Database:** PostgreSQL for data management.<br>
**Cloud:** GCP or AWS.<br>
**Data Streaming:** Kafka for real-time events, and Spark for batch processing tasks.<br>
